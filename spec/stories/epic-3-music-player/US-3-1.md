## Mô tả

```
Là một người dùng,
Tôi muốn bật/tắt nhạc nền,
Để tôi có thể tập trung với âm nhạc phù hợp.
```

## Tiêu chí chấp nhận

- [x] Nút play (▶) / pause (❚❚) ở thanh điều khiển dưới màn hình
- [x] Thanh điều khiển fixed ở bottom, center
- [x] Hiển thị tên bài đang phát (capitalize, bỏ prefix "freepik-")
- [x] Text "Music off" khi không phát (màu mờ)
- [x] Tự động phát bài tiếp theo khi hết (audio `ended` event)
- [x] Playlist shuffle sau mỗi vòng lặp

## Technical Notes

- `<audio id="audioPlayer">` element
- `toggleMusic()` function
- State: `musicOn`, `shuffledPlaylist`, `currentTrackIndex`
- `audioPlayer.addEventListener('ended', skipTrack)`
