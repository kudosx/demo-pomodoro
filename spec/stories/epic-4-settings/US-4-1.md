# US-4.1: Mở/đóng Settings

**Epic:** [Settings](EPIC.md)
**Trạng thái:** ✅ Done
**Độ ưu tiên:** High

---

## Mô tả

```
Là một người dùng,
Tôi muốn truy cập cài đặt qua nút ⚙️,
Để tôi có thể tùy chỉnh trải nghiệm sử dụng.
```

## Tiêu chí chấp nhận

- [x] Nút Settings (⚙️) fixed góc trên phải màn hình
- [x] Hover có animation xoay 45°
- [x] Click mở modal overlay (backdrop blur)
- [x] Modal chiếm 90vw x 80vh, max-width 1100px
- [x] Click overlay (bên ngoài modal) để đóng
- [x] Nút X góc trên phải modal để đóng
- [x] Dark theme UI (background rgba(30,30,30,0.95))

## Technical Notes

- `.settings-overlay.active` để hiển thị
- `openSettings()`, `closeSettings()`, `closeSettingsOverlay(event)`
- CSS transition cho fade effect
- `backdrop-filter: blur(5px)` cho overlay
