# US-2.2: Timeline trong ngày

**Epic:** [Progress Tracking](EPIC.md)
**Trạng thái:** ✅ Done
**Độ ưu tiên:** High

---

## Mô tả

```
Là một người dùng,
Tôi muốn xem timeline các phiên đã hoàn thành trong ngày,
Để tôi có cái nhìn tổng quan về hoạt động của mình.
```

## Tiêu chí chấp nhận

- [x] Hiển thị các block theo thứ tự thời gian (sớm → muộn)
- [x] Mỗi block hiển thị:
  - Thời gian (HH:MM format, 24h)
  - Loại phiên: W (Work), SB (Short Break), LB (Long Break)
- [x] Màu sắc phân biệt:
  - Work: #e74c3c (đỏ)
  - Short Break: #3498db (xanh dương)
  - Long Break: #9b59b6 (tím)
- [x] Hover hiển thị tooltip với tiêu đề và thời gian đầy đủ
- [x] Legend ở dưới giải thích màu sắc
- [x] Block có animation scale khi hover

## Technical Notes

- localStorage key: `pomodoroTimeline`
- Data structure:
  ```javascript
  {
    date: "Mon Dec 02 2024",
    sessions: [
      { mode: "work", time: "14:30", title: "Task name" },
      ...
    ]
  }
  ```
- `renderTimeline()` generates HTML
- CSS flexbox với wrap cho responsive
