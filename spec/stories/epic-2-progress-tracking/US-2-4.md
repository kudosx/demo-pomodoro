**Epic:** [Progress Tracking](EPIC.md)

**Độ ưu tiên:** Low

## Mô tả

```
Là một người dùng,
Tôi muốn xóa timeline của ngày,
Để tôi có thể bắt đầu lại khi cần.
```

## Tiêu chí chấp nhận

- [x] Nút "Clear Timeline" ở dưới timeline section
- [x] Click hiện confirmation dialog: "Clear today's timeline?"
- [x] Confirm xóa:
  - Xóa localStorage data
  - Reset session count về 0
  - Update DOM
- [x] Timeline hiển thị: "No sessions yet. Start your first Pomodoro!"

## Technical Notes

- `clearTimeline()` function
- `localStorage.removeItem('pomodoroTimeline')`
- Uses native `confirm()` dialog
