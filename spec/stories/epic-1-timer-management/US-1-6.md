# US-1.6: Cấu hình thời gian phiên làm việc

**Epic:** [Timer Management](EPIC.md)
**Trạng thái:** ✅ Done
**Độ ưu tiên:** High

---

## Mô tả

```
Là một người dùng,
Tôi muốn chỉnh thời gian của phiên làm việc hiện tại,
Để tôi có thể linh hoạt điều chỉnh theo nhu cầu cụ thể.
```

## Tiêu chí chấp nhận

- [x] Click vào timer display để chuyển sang chế độ edit
- [x] Hiển thị input cho phép nhập thời gian (phút)
- [x] Nhấn Enter hoặc blur để xác nhận
- [x] Nhấn Escape để hủy chỉnh sửa
- [x] Chỉ cho phép edit khi timer đang dừng (không running)
- [x] Validation: 1-60 phút

## Technical Notes

- Thêm click handler cho `#timer` element
- Tạo input overlay hoặc thay thế display tạm thời
- Cập nhật `timeLeft` và gọi `updateDisplay()`
