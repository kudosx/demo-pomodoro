# US-1.4: Chuyển đổi chế độ

**Epic:** [Timer Management](EPIC.md)
**Trạng thái:** ✅ Done
**Độ ưu tiên:** Critical

---

## Mô tả

```
Là một người dùng,
Tôi muốn chuyển giữa các chế độ Work/Short Break/Long Break,
Để tôi có thể linh hoạt quản lý thời gian làm việc và nghỉ ngơi.
```

## Tiêu chí chấp nhận

- [x] 3 nút mode với thời gian cấu hình:
  - Work: 25 phút
  - Short Break: 5 phút
  - Long Break: 15 phút
- [x] Click chuyển mode và reset thời gian tương ứng
- [x] Nút active có style highlight (background đỏ, text trắng)
- [x] Hiển thị label: "Work Time", "Short Break", "Long Break"
- [x] Nếu đang chạy, tự động pause khi chuyển mode

## Technical Notes

- `MODES` object chứa config
- `setMode(mode)` xử lý chuyển đổi
- CSS class `.active` cho button styling
