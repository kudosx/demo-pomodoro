# US-1.1: Khởi động phiên làm việc

**Epic:** [Timer Management](EPIC.md)
**Trạng thái:** ✅ Done
**Độ ưu tiên:** Critical

---

## Mô tả

```
Là một người dùng,
Tôi muốn bắt đầu bộ đếm thời gian 25 phút,
Để tôi có thể tập trung làm việc trong một khoảng thời gian xác định.
```

## Tiêu chí chấp nhận

- [x] Hiển thị đồng hồ đếm ngược 25:00 (format MM:SS)
- [x] Nút "Start" khởi động bộ đếm
- [x] Nút chuyển thành "Pause" khi đang chạy
- [x] Thời gian đếm ngược chính xác từng giây (setInterval 1000ms)
- [x] Hiển thị nhãn "Work Time" khi ở chế độ làm việc
- [x] Font số sử dụng tabular-nums để không nhảy layout

## Technical Notes

- Timer state: `isRunning`, `timeLeft`, `interval`
- Mode config trong `MODES.work.time = 25 * 60`
- Display update qua `updateDisplay()` function
